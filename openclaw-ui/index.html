使用逻辑

建议这样用

  1. 先刷新状态，确认网关、Agent 和工作区都正常。
  2. 在“发布任务”里直接写中文要求，结果会自动记入“最近任务”。
  3. 要复用旧任务时，在“最近任务”里一键回填或重新发送，不用重新组织措辞。
  4. 只有在需要看官方聊天界面时，再使用右上角的 Dashboard 入口。

稳定性说明

这台机器的 Windows 到 WSL 官方网关连接不稳定，所以本地任务中心被设计成主入口。它直接调用 OpenClaw CLI，不依赖官方页 websocket。
