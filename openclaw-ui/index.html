使用逻辑

你可以这样用

  1. 先刷新状态，确认 OpenClaw 网关、工作区和分支信息都正常。
  2. 需要查环境时，先跑“项目体检”或测试按钮，快速验证基础链路。
  3. 需要让 OpenClaw 帮你分析任务时，在下方 Agent 区直接输入中文要求。
  4. 最近视频、常用命令和运行日志会把结果串起来，方便继续下一步。
