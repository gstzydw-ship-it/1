使用逻辑

建议这样用

  1. 先刷新状态，确认网关、工作区和当前分支都正常。
  2. 如果官方 Dashboard 红字报错 `token mismatch`，先点“复制带令牌链接”，再重新打开页面。
  3. 需要做项目检查时，用右侧快捷操作；需要让 OpenClaw 帮你分析任务，就在下方 Agent 区直接下发中文指令。
  4. 生成结果统一看“最近视频”和“运行日志”，不需要来回切页面。
